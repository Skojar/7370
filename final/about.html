Theme Switcher

The theme switcher utilizes modular external style sheets. One with basic structural styling is always linked. The other, containing color, font, background images urls, and other theme-specific styling is in a set of additional style sheets, one for each theme. They're traded out using JQuery..

Rules Reference

Each game has an external html file with navigation items, citation text, and individual rules, portions of which are loaded dynamically with AJAX through the dropdown menus and displayed in a simple lightbox. This list of features works the same way.

Dice Roller

This is a straightforward javascript dice function, using images, css, and unicode characters to generate PbtA-style dice results:

  • ✖ = Failure
  • ❓ = Partial Success
  • ★ = Success
  • ♛ = Advanced Success

Move Suggester

Another basic javascript function, selecting a random element from an array and displaying it, either on demand or on a timer. The only vaguely interesting part is that the suggestions won't immediately repeat.

Navigation

Drop-down navigation menus are closely modeled after suckerfish dropdowns.
